My Projects

NMSU-A: DesignPlus HTML Parser

Description:

The DesignPlus HTML Parser is a tool I designed to solve a very specific issue my department had while working on online courses. There was an issue with working with a legacy version of DesignPlus on Canvas where Online Learning was trying to change branding for the online courses, but there wasn't an easy way to implement complex styling at an efficient speed. I used principles I learned in my Python class and built a prototype to help me speed up adding the styling to NMSU-A courses. After I built a working prototype, I brought the concept to my boss and began working on a tool that my whole department could use without needing a comprehensive understanding of HTML and CSS.  

Key Features:

  • Formats clipboard contents for easy pasting back into Canvas.
  • Ensures H2 headers are properly structured and inserts progress bars where needed.
  • Applies predefined class attributes for uniform styling.
  • Removes redundant progress bars and correctly formats hidden elements.
  • Built using Python 3.11

Python Libraries Used:

  • Pyperclip
  • BeautifulSoup4
  • CustomTkinter
  • Tkinter MessageBox

HeadSynk - A Behavioral Health Tracker

Description:

HeadSynk is a Python-based application developed for CS 153: Python Programming at New Mexico State University Alamogordo. It is designed to track and analyze personal data over two weeks, featuring tools for tracking mood, food intake, water consumption, sleep, exercise, and journaling, helping users maintain consistency and meet their wellness goals. Developed by Zachary A. Carmichael and Shabhan Andrew George with contributions from Raena Martinez, William Jones, and Jason Edgington.

Key Features:

  • Mood Tracker: Allows users to rate their daily mood on a scale from 1 to 5 and offers motivational messages.
  • Food Tracker: Logs intake of major food groups and encourages balanced eating habits.
  • Water Tracker: Tracks daily water intake and provides feedback to meet recommended goals.
  • Sleep Tracker: Logs hours of sleep and advises users for optimal rest.
  • Journal: A space for users to write down daily notes.
  • Badge System: Displays a badge image (unfinished).
  • Data Persistence: Saves data in 'user_data.json' with nested dictionaries.

Libraries Used:

  • CustomTkinter
  • PIL (Python Imaging Library)
  • Datetime
  • JSON

ByteBungalow

Description:

A webscraper and database to help you in your search for rental listings in any given city.
The idea for this project is to be a one-stop database for you to quickly look at all listings in your area from some of the most popular websites by ethically scraping publically accessible data for non-commercial purposes.
This project was made purely as a educational effort for the CS 272 class at NMSU-A.

Key Features:

  • The webscraper grabs publicly available listings data from rental sites.
  • BBDatabase is the custom database we use built on sqlite3 to store the scraped data.
  • BBsearch lets us sort through the data based on a listing object that queries the database based on shared attributes.
  • Data visualization uses mock data to give graphs based on site rent and rooms.
  • The Listings object is a custom object type that allows us to store attributes such as: name, rent, number of rooms, whether utilities are included, whether you have favorited the listing or not, etc.

Libraries Used:

  • Dash
  • Matplotlib
  • SQLite3
  • Requests
  • BeautifulSoup4
  • Playwright
